Part 1: data only

Part 2: executable program populates data structure with K black holes placed in random locations and display game board

Part 3: executable program populates data structure with K black holes placed in random locations, computes and stores the number of adjacent black holes and displays game board

Part 4: executable program allows to set game settings (board size and number of black holes) and play the game using console input.
 Also it supports additional command line arguments:
 -h - help;
 -d - debug;
 -f - read game board from the specified file
